(Eine kurze Beschreibung des Projekts)
erstesProjekt

Dieses Projekt enthält die Analyse für XY-Datensatz. Das Ziel des Projekt ist die Entwicklung eines Modells um Annahme-Ablehnung vorherzusagen. Die Erklärbarkeit des Algorithmus spielt in diesem Projekt eine Rolle.

Die Analyse ist ausschließlich in Python durchgeführt.



Installationsanweisungen / Abhängigkeiten

Dieses Projekt arbeitet an der Conda-Umgebung.
    1. Installieren Sie zunächst die Anaconda Python-Distribution.
    2. Neue Conda-Umgebung erstellen, aktivieren und installieren sie nötige Bibliotheke mit
          
              conda install --file requirements.txt
        
    3. Überprüfen Sie die Numpy, Pandas und Matplotlib-Bibliotheken


Inhaltsverzeichnis

    1. data
        * data.csv: XY-Datensatz, Komma-separiert.
    2. module
        * neben_funktionen.py: Das Python-Script enthält Funktionen, um die Daten schnell zu visualisieren.
    3. main.ipynb: Die Haupt-Notebook des Projekts. Dieses Notebook sollte zur Steuerung des Projekts ausgeführt werden.
    4. requirements.txt: Die Datei beinhaltet die angewendeten Python-Bibliotheke und ihre Versionen.
